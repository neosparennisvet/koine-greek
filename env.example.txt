NEXT_PUBLIC_SUPABASE_URL=https://XXXXXXXX.supabase.co

SUPABASE_SERVICE_ROLE_KEY=eyXXXXXXX...

Где взять ключи:
- SUPABASE_URL и ключи: supabase.com → ваш проект → Settings → API
- YOOKASSA: yookassa.ru → Настройки → Ключи API
- SITE_URL: пока оставьте http://localhost:3000
  После деплоя на Vercel замените на https://ваш-сайт.vercel.app
