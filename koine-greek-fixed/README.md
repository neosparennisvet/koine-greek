# Κοινή Greek — Курс библейского греческого языка

Полноценный сайт с регистрацией, оплатой через ЮКассу и личным кабинетом студента.

---

## 🚀 Пошаговый запуск (без технических знаний)

### Шаг 1 — Установите Node.js на компьютер
1. Откройте сайт https://nodejs.org
2. Скачайте версию **LTS** (зелёная кнопка)
3. Установите как обычную программу

### Шаг 2 — Зарегистрируйтесь на GitHub
1. Откройте https://github.com и нажмите **Sign up**
2. Создайте аккаунт (бесплатно)
3. Создайте новый репозиторий: **New** → назовите `koine-greek` → **Create repository**
4. Загрузите папку с кодом в репозиторий

### Шаг 3 — Создайте базу данных в Supabase
1. Откройте https://supabase.com → **Start your project** (бесплатно)
2. Создайте новый проект (выберите регион **Frankfurt** или **Singapore**)
3. Дождитесь запуска (~2 минуты)
4. Откройте **SQL Editor** → **New query**
5. Скопируйте содержимое файла `supabase/migrations/001_schema.sql` и нажмите **Run**
6. Перейдите **Settings → API** и скопируйте:
   - `Project URL` → это `NEXT_PUBLIC_SUPABASE_URL`
   - `anon public` ключ → это `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `service_role` ключ → это `SUPABASE_SERVICE_ROLE_KEY`

### Шаг 4 — Подключите ЮКассу
1. Зарегистрируйтесь на https://yookassa.ru
2. Пройдите верификацию (нужен ИП или ООО)
3. Перейдите **Настройки → Ключи API**
4. Скопируйте **shopId** и **Секретный ключ**

### Шаг 5 — Разверните сайт на Vercel
1. Откройте https://vercel.com → **Start Deploying** → войдите через GitHub
2. Нажмите **Import** → выберите репозиторий `koine-greek`
3. Перед деплоем нажмите **Environment Variables** и добавьте:

```
NEXT_PUBLIC_SUPABASE_URL        = ваш URL из шага 3
NEXT_PUBLIC_SUPABASE_ANON_KEY   = ваш anon ключ
SUPABASE_SERVICE_ROLE_KEY       = ваш service_role ключ
YOOKASSA_SHOP_ID                = ваш shopId
YOOKASSA_SECRET_KEY             = ваш секретный ключ
NEXT_PUBLIC_SITE_URL            = https://ваш-домен.vercel.app
NEXT_PUBLIC_COURSE_PRICE_RUB    = 14900
```

4. Нажмите **Deploy**. Сайт запустится через ~3 минуты.

### Шаг 6 — Настройте вебхук ЮКассы
После деплоя вернитесь в ЮКассу:
1. **Настройки → HTTP-уведомления**
2. Укажите URL: `https://ваш-домен.vercel.app/api/payments/yookassa`
3. Отметьте событие: `payment.succeeded`

## 🛠 Локальная разработка

```bash
# Установить зависимости
npm install

# Скопировать переменные окружения
cp .env.example .env.local
# Заполните .env.local своими ключами

# Запустить сайт локально
npm run dev
# Откройте http://localhost:3000
```

## ➕ Как добавить контент уроков

Каждый урок имеет ID вида `1-1`, `2-3` и т.д. Для добавления контента:

1. Создайте файл `/lib/lessons/2-2.ts`
2. Экспортируйте объект типа `LessonContent` (см. `/types/index.ts`)
3. Импортируйте в компонент `LessonContent.tsx`

Полный контент для всех уроков добавляется последовательно.
